Instructions:

  • Upload Image: Click the "Upload Image" button to add an image to the canvas.
  • Draw: Select a color and size and start drawing on the canvas.
  • Eraser: Click "Eraser" to erase your drawing.
  • Clear Canvas: Clears the current canvas but keeps the image.
  • Add Tabs: Click the "+" button to add more canvas tabs.
  • Download Canvas: Click "Download Canvas" to save the current canvas as a PNG file.
